## ALGORITHM:

### Step 2:

### Step 3:

### Step 4:

### Step 5: 
End the program.
### PROGRAM:

    import math
  
    x1, y1 = 4, 2

    x2, y2 = 10, 6

    distance = math.sqrt((x2 - x1)**2 + (y2 - y1)**2)

    print("{:.2f}".format(distance))
